# C#的类型系统
C# 采用统一的类型系统，因此任意类型的值都可视为 object。 每种 C# 类型都直接或间接地派生自 object 类类型，而 object 是所有类型的最终基类。 只需将值视为类型 object，即可将引用类型的值视为对象。 通过执行装箱和取消装箱操作，可以将值类型的值视为对象。
